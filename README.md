# Installatron Translator Source Files

This repository contains the source files used by the Installatron Translator tool.

## Contributing

Please submit all changes and updates through the Installatron Translator tool.  
Do **not** fork this repository or submit changes via pull requests, as they will not be accepted.

## Purpose

This repository exists solely to support the operation of the Installatron Translator tool and is not intended for direct development workflows.
